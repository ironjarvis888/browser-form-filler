# Browser Form Filler | 網頁表單自動填寫

## Description | 描述

Automatically analyze and fill web forms. 自動分析並填寫網頁表單。

## Features | 功能

- Form structure analysis | 表單結構分析
- Parse input fields | 解析輸入欄位
- Parse select dropdowns | 解析下拉選單
- Auto-submit forms | 自動提交表單

## Installation | 安裝

```bash
pip3 install requests beautifulsoup4
```

## Usage | 使用方式

```bash
# Analyze form | 分析表單
python3 fill_form.py https://example.com/form

# Fill and submit | 填寫並提交
python3 fill_form.py https://example.com/form name=John email=john@example.com
```
